## 자격증 선정

SAP-C02를 취득하고 나서 다른 자격증을 무엇을 취득할까 고민을 많이 했습니다. Professional 위에 Specialty라는 자격증이 있다는 것을 보고 클라우드 서비스를 사용하는 데 보안이 빠질 수 없다고 생각이 들어서 Security - Specialty를 준비하게 되었습니다.

## Specialty 준비

자격증 준비를 위해 강의는 이제는 빠지면 섭한 Stephane Maarek 강사님의 [Ultimate AWS Certified Security Specialty [NEW 2026] SCS-C02](https://www.udemy.com/course/ultimate-aws-certified-security-specialty/)를 들으며 준비했습니다. 

![또판 Maarek 강의를 들으며 준비..](/images/CSP/AWS/getting-security-specialty/28f9c25b-d4e6-426e-a151-95f7c176c8e3-image.png)

그래도 평소에 AWS를 사용하면서 보안이나 Best Practice를 Docs에 찾아보면서 했었기에, 내용 자체는 익숙해서 3주 정도 준비하고 시험을 쳤습니다. 다만 KMS 관련해서 Envelop Encryption이나 Cross-Account Access 와 같은 생소한 개념이 있어서 해당 부분은 좀 집중적으로 보면 도움이 될 것 같습니다.

자격증 가격은 $300 달러로 Professional와 동일한 금액이었기에... 이번에도 한번에 따고 싶었던 희망이 있었습니다.
